When & How to Use Each Color

Expert guidance on applying each color for maximum impact and consistency.

Deep Navy #0D1B5E Use at ~40% of the design

  • Navigation bar background
  • Page footers and dark sections
  • Hero section backgrounds
  • Modal overlays & sidebars
  • Text color on light backgrounds for headings
  • Primary button background when gold is unavailable

Royal Navy #1A2B7A Use at ~15% of the design

  • Gradient midpoint (Deep → Steel Navy)
  • Hover state for dark nav links
  • Section header pill/badge backgrounds
  • Timeline or stepper active color
  • Focus ring on dark surfaces

Steel Navy #3C508C Use at ~10% of the design

  • Gradient end / lighter dark surfaces
  • Icon button backgrounds
  • Tag / chip borders on dark UI
  • Secondary text on dark cards
  • Dividers within dark sections

Expo Gold #C8A064 Use at ~15% — the hero accent

  • Primary CTA buttons ("Register Now", "Explore")
  • Active nav indicator / underline
  • Highlighted text spans & key stats
  • Card top-border accent stripe
  • Logo accent color
  • Star ratings, award icons
  • Section label pills on dark backgrounds

Warm Amber #B48C50 Interaction / depth only

  • Hover & pressed state for gold buttons
  • Darker gold text on gold backgrounds
  • Link hover color in body text
  • Decorative border-bottom on headings
  • Form focus ring on light fields

Sand Tint #DCC8A0 Backgrounds & cards

  • Info/announcement card backgrounds
  • Alternating table row shading
  • Tooltip & popover backgrounds
  • Input field background on navy sections
  • Subtle section background variation

Ivory White #F5F4EF The breathing space (~20%)

  • Main page / app background
  • White card bodies
  • Form field backgrounds
  • Text on navy: always white/ivory
  • Spacing between dense sections

Ready-to-Use CSS Variables

Copy this into your project's global stylesheet or Tailwind config.

/* ── EduGate Brand Palette ─────────────────────────── */
:root {
  /* Navy family */
  --color-navy-deep:    #0D1B5E;   /* primary brand, navbars, footer */
  --color-navy-mid:     #1A2B7A;   /* gradient midpoint, hover states */
  --color-navy-light:   #3C508C;   /* gradient end, icons */

  /* Gold family */
  --color-gold:         #C8A064;   /* CTAs, highlights, badges */
  --color-gold-dark:    #B48C50;   /* hover / pressed states */
  --color-gold-pale:    #DCC8A0;   /* card & section backgrounds */

  /* Neutrals */
  --color-bg:           #F5F4EF;   /* page background */
  --color-white:        #FFFFFF;   /* card surfaces */
  --color-text:         #0D1B5E;   /* body text on light bg */
  --color-text-muted:   #5A6480;   /* secondary text */
}

/* ── Tailwind theme extension ───────────────────────── */
/* tailwind.config.js → theme.extend.colors: */
navy: {
  DEFAULT: '#0D1B5E',
  mid:     '#1A2B7A',
  light:   '#3C508C',
},
gold: {
  DEFAULT: '#C8A064',
  dark:    '#B48C50',
  pale:    '#DCC8A0',
},

The Golden Rule: 60 – 30 – 10

A proven distribution formula for the EduGate palette.

60%
Navy Blues
Navbars, footers, hero sections, backgrounds. The dominant brand colour that establishes trust & authority.
30%
Ivory & Sand
Page backgrounds, card surfaces, form fields. Provides visual rest and contrast against the navy.
10%
Expo Gold
CTAs, highlights, icons, badges. Used sparingly so it commands attention every single time it appears.
